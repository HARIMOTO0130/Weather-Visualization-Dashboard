# 项目环境配置要求

## 1. 前端环境

### Python环境
- Python 3.8+
- 依赖包安装：
  ```
  pip install -r 前端/requirements.txt
  ```

### Django配置
- Django 4.1.7
- 数据库配置已设置为MySQL，连接信息：
  - 用户名：root
  - 密码：1234
  - 主机：192.168.56.101
  - 端口：3306
  - 数据库名：china_all

### 其他前端依赖
- ECharts.js (通过CDN引入)
- jQuery (通过CDN引入)

## 2. 后端环境

### Java环境
- JDK 1.8
- Maven 3.6+

### Hadoop环境
- Hadoop 2.9.2
- 配置项：
  - core-site.xml
  - hdfs-site.xml
  - mapred-site.xml
  - yarn-site.xml

### HDFS路径配置
- 输入数据路径：hdfs://master:9000/china_all/
- 输出数据路径：
  - 最高温度：hdfs://master:9000/output/maxtemp/
  - 最低温度：hdfs://master:9000/output/mintemp/

## 3. 数据库环境

### MySQL
- 版本：5.7+
- 虚拟机地址：192.168.56.101:3306
- 数据库名：china_all
- 用户：root
- 密码：1234
- 字符集：utf8mb4

## 4. 系统要求
- Linux/MacOS/Windows均可运行前端
- 后端MapReduce任务需要在Hadoop集群环境中运行
- 建议至少8GB内存
- 建议至少50GB存储空间
